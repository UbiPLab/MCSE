ID TRUST
AB Cloud computing is a hotspot in the Internet since the 21st century, and it is a new model suitable for mobile Internet applications. But the technology has not been widely applied in the economic resource management. Based on the current economic model, this paper designs the cloud economic resource management model, and analyzes the function of the main component cloud market. In addition, this paper gives the specific process of cloud market transactions, and describes a number of important mechanisms for resource management in detail. Finally, this paper improves the traditional reputation evaluation function and proposes a model of cloud resource reputation evaluation, which provides important reference value for the future development of cloud technology.
